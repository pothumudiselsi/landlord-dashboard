# Landlord Dashboard - Java Console Application

The Landlord Dashboard is a simple console-based Java application designed to help landlords manage rental properties efficiently. 

This application allows users to:
- Add new properties with rent and occupancy status
- View all stored properties
- Display a dashboard summary including total properties, occupied properties, vacant properties, and total rental income

## Key Features
- Object-Oriented Programming (OOP) implementation
- Dynamic data storage using ArrayList
- User input handling with Scanner
- Dashboard analytics summary
- Clean menu-driven interface

## Technologies Used
- Java
- ArrayList
- Scanner Class
- OOP Concepts (Encapsulation & Objects)

## Learning Outcomes
This project demonstrates:
- Basic Java programming
- Class and object creation
- Data handling using collections
- Conditional logic and loops
- Menu-driven console application development
